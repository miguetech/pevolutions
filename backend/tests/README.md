# 🧪 Tests - PEvolutions API

Archivos de prueba para todos los endpoints de la API.

## 📁 Contenido

- **`test_all_endpoints.py`** - Script Python automatizado (recomendado)
- **`api-tests.http`** - Para REST Client (VSCode)
- **`thunder-collection.json`** - Para Thunder Client (VSCode)
- **`quick-test.sh`** - Script bash con curl
- **`TESTING_GUIDE.md`** - Guía completa de uso

## 🚀 Uso Rápido

```bash
# Desde la raíz del proyecto backend/
python tests/test_all_endpoints.py
```

## 📚 Documentación

Ver `TESTING_GUIDE.md` para instrucciones detalladas de cada método de prueba.
